# Rummikub — AI Strategy Game

A full-featured Rummikub game with multiple AI strategies and human multiplayer support.

## Features
- 2–4 players, mix of human and AI
- 4 AI strategies: Aggressive, Defensive, Balanced, Speed
- Full rule enforcement: valid sets, initial meld ≥30, board rearrangement
- 106-tile pool with jokers

## Quick Start
```bash
npm install
npm run dev
```

## Deploy to Vercel
```bash
git init && git add . && git commit -m "Rummikub"
gh repo create rummikub --public --push
# Then import on vercel.com
```
